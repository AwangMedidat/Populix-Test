# Movie App

## Getting Started

1. Clone the repository: `cd MovieApp/`.

2. Install dependencies: `npm install`

3. Run the development server:

if you Windows OS:

```bash
npx react-native run-android
```

if you iOS:

```bash
npx react-native run-ios
```

## Additional Started
